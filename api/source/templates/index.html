Welcome to Everything Avocados!

Navigate between the different pages/forms in the header to submit different requests to our api. These requests are interacting with multiple kubernetes clusters that contain worker nodes to do tasks that involve intercating with a Redis database as well as a Postgres database. This HTML site is brought to you by a NodePort service on TACC's ISP machine that is currently pointing at our flask app cluster.
